Система  HotelWEB предназначена для
•	Автоматизация учета постояльцев в гостинице;
•	Заселение, обеспечения проживания и организация выезда клиентов;
•	Планирование проживания;
•	Получение отчетной информации.

Выполняемые функции:

•	Механизм бронирования номера  как в ручном режиме  оператором/  так режим приема бронирования через  web форму на сайте ( отправка заявок оператору системы и их автоматическая обработка  / рассылка уведомлений по почте).
•	Заселение постояльцев;
•	Управление гостями ( добавление / удаление / изменение )-  можно добавление картинок к карточке гостя ( скан пасспорта )
•	Личный счет гостя / учет оказанных услуг за какой то период
•	Обслуживание проживающих,  учет дополнительных услуг;
•	Организация выезда, учет выбывших;
•	Ведение  учета постоянных клиентов и скидки для них ( 5% 10%) 
•	Ведение архива проживавших в гостинице;
•	Ведение справочника услуг гостиницы;
•	Формирование, печать  счетов за услуги и проживание;
•	Формирование отчетов за любой период;
•	Выдача справочной информации о местах в гостинице в графическом виде;
•	Встроенная поисковая система;
•	Выдача информации текущей брони, проживающих, выезжающих, долгах по услугам.
•	Работа с номерами, учет их использования, замечания по ним. Учет уборки номеров.
•	Защита данных от неверных действий пользователя

Поддержка :
•	Роли Администратор ( управление всей системой и структурой комнат, наличием  услуг , сотрудников их ролей  и тд. / Функционал печати отчетов ( за месяц по указанным услугам и тд ..) / export import базы / возможно настройка бекапов ..
•	Роль Оператор ( управление заказами создание нового , редактирование текучего , выезд гостя / печать  чека по форме )
•	Роль Гость – для online бронирования заказа
